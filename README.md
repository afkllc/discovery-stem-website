## Overview

This site is the web presence for Discovery STEM Education. It showcases our programs, events, and resources using Astro with Tailwind CSS and TypeScript. The content is managed via Markdown files located in `src/content`.

### Develop & build

```bash
npm install
npm run dev        # start dev server
npm run build      # produce production output in dist/
```

The site is fully static and works without external dependencies.
