## Overview

This site is the web presence for Discovery STEM Education. It showcases our programs, events, and resources using Astro with Tailwind CSS and TypeScript. The content is managed mostly via Markdown files in `src/content`.

### Develop & build

```bash
npm install
npm run dev        
npm run build      
```

Site is currently running at: [Discovery STEM Website](https://discovery-stem.netlify.app/)
